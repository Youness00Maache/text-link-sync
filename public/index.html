Built for the useful stuff

More direct than sending it to yourself.

TextLinker keeps the transfer flow focused: connect, choose the item, and move on.

  • Text that stays usableOpen notes immediately, copy them, and follow detected links or email addresses.
  • Files on demandBrowse the phone library first, then request only the file you actually need.
  • Send back to AndroidMove one text or one file up to 25 MB from the browser to the connected phone.

Short-lived by design

A transfer session, not another account.

Each QR creates a temporary session. Requests are limited, file sizes are capped, and the browser checks that messages belong to the active pairing.

No forced sign-in

Connect the devices you have in front of you without creating a profile.

10-minute pairing

The QR session expires instead of remaining open indefinitely.

Request limits

One active file request at a time, with cooldowns to reduce accidental or abusive spam.

Validated transfers

Session ID, pairing token, expiry, message type, and file size are checked before use.
