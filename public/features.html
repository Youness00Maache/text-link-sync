Features

The useful path between your phone and computer.

TextLinker keeps the workflow narrow and practical: connect quickly, see what is available, and move only the text or file you need.

Android to browser

Your phone library, organized for the web.

The Android app shares a structured manifest, not a pile of files. The website renders direct folders, notes, and file metadata.

Folder navigation

Open nested groups, move back through their hierarchy, and see only direct children at each level.

Instant text notes

Full note content opens immediately with copy controls and automatic link and email detection.

File metadata first

Inspect name, type, size, and date before asking the phone to upload the actual file.

File experience

Request, preview, open, or download.

TextLinker chooses a real icon and a suitable viewer based on the file's MIME type.

Media previews

Images, video, and audio use suitable browser preview and playback controls when received.

Documents and PDF

PDF and text formats open in an appropriate viewer, while office files retain recognizable document icons.

Private-looking opens

The browser fetches the file as a temporary blob before opening or downloading, avoiding storage URLs in the address bar.

Browser to Android

Send something back without starting over.

Once connected, use the send action beside the library counters to choose text or a file.

Send text

Add a title, paste the content, and send the full note through the active pairing.

Send one file

Choose one file up to 25 MB. Larger uploads use the resumable transfer path.

Stay connected

The receiver keeps listening during the session instead of closing after the first library message.

Get the Android app

TextLinker starts on the phone you already use.

Install the Android app from Google Play, then use the web receiver from any supported desktop browser.

QR pairing

No username or password needs to be typed on the computer.

Validated sessions

Pairing token, session ID, expiry, payload kind, counts, and file sizes are checked.

Temporary access

Sessions expire after about 10 minutes and file requests are rate-limited.
